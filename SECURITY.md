# Security Policy

## Supported Versions

Only the latest version of Zonnestroom 2.0 is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| < 2.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately. Do **not** create a public issue.

Please send an email to [wmostert76@gmail.com](mailto:wmostert76@gmail.com) with:
- A description of the vulnerability.
- Steps to reproduce it.
- Possible impact.

We will acknowledge your report within 48 hours and provide updates on our progress.
